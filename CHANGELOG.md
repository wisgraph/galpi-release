# Galpi Change Log Archive

## v1.0.6
### v1.0.6 업데이트 안내 🚀

이번 업데이트에서는 검색 엔진의 정확도를 정밀하게 조정하고, 사용자님의 설정에 맞춰 런처 창의 크기와 위치를 유지하는 기능을 추가했습니다.

#### 📐 창 크기 및 위치 기억 (Dynamic Persisting)
- **개별 맞춤 리스트 설정**: 설정에서 노출될 갈피 개수(6~12개)를 조정하면, 해당 개수에 딱 맞게 런처의 세로 길이가 자동으로 최적화됩니다. 
- **위치 기억 기능**: 드래그 모드를 사용 중일 때, 런처를 이동시킨 위치를 기억합니다. 이제 매번 창을 켤 때마다 중앙으로 복구되지 않고, 최근에 배치한 그 위치에서 바로 나타납니다.
- **메뉴바 상시 구동 최적화**: macOS 하단 Dock에서 아이콘이 노출되지 않도록 개선했습니다. 이제 레이캐스트(Raycast)처럼 메뉴바 아이콘만으로 깔끔하게 상시 구동됩니다.
- **기본 단축키 변경 (Conflict 방지)**: 기존 `Option + Space`에서 `Option + Command + G`로 기본 단축키를 변경했습니다. 이제 macOS 시스템 단축키나 다른 앱(Spotlight, Raycast 등)과의 충돌 걱정 없이 바로 사용하실 수 있습니다.
- **공간 활용 최적화**: 더 적은 항목을 볼 때는 컴팩트하게, 많은 정보를 볼 때는 시원하게 창 크기가 조절되어 더 높은 몰입감을 제공합니다.

#### 🔍 검색 및 하이라이트 시스템 고도화
- **검색 정확도 원천 복구**: 가장 안정적이었던 검색 엔진 로직으로 회귀하여, "ㄴ" 한 글자 초성이나 "dk" 같은 짧은 영어 키워드도 누락 없이 완벽하게 찾아냅니다.
- **하이브리드 하이라이트 안정화**: 백엔드의 강력한 정렬 성능과 프론트엔드의 정교한 하이라이트 범위를 결합하여, 검색 결과가 훨씬 깔끔하고 의도에 맞게 강조됩니다.
- **태그 자동완성 필터링 개선**: 태그 검색 시 발생하던 필터링 오류를 해결하여, 타입 지정 없이도 정확하게 태그 기반 검색이 가능하도록 개선했습니다.

#### 🛠️ 시스템 안정성 개선
- **검색 임계값 제거**: 인위적인 점수 컷오프를 없애고 모든 유효한 결과를 노출하여 검색 결과의 신뢰도를 높였습니다.
- **버전 동기화 최적화**: 내부 라이브러리 간의 버전 및 타입 일치를 통해 시스템 안정성을 강화했습니다.

항상 갈피를 더 편리하게 만들어주시는 사용자 여러분의 의견에 감사드립니다! 😊

## v1.0.5
### v1.0.5 업데이트 안내 🚀

이번 업데이트에서는 더욱 쾌적한 정보 확인을 위해 런처의 규격을 개선하고 디자인 디테일을 조정했습니다.

#### 🎨 디자인 및 사용성 개선
- **런처 크기 확대**: 런처 창의 크기를 약 10% 확대하여 한 번에 더 많은 정보를 시원하게 확인할 수 있도록 개선했습니다. (770x500)
- **태그 가독성 향상**: 리스트 내 태그의 글자 크기를 키우고 여백을 조정하여 시인성을 높였습니다. 
- **업데이트 시스템 완결**: 새롭게 도입된 프리미엄 업데이트 UI와 자동 업데이트 흐름이 이번 버전부터 완벽하게 적용됩니다.

항상 갈피를 응원해 주셔서 감사합니다! 😊

## v1.0.4
### v1.0.4 업데이트 안내 🚀

이번 업데이트에서는 캡처 카드 기능의 완성도를 높이고, 키보드 네비게이션과 시스템 안정성을 획기적으로 개선했습니다.

#### ✨ 주요 기능 및 개선 사항
- **캡처 카드 컨텍스트 메뉴 강화**: 이제 캡처된 카드에서 바로 수정, 열기, 복사, 삭제 등의 액션을 수행할 수 있습니다. (Meta+k 단축키 지원)
- **키보드 네비게이션 최적화**: 검색 결과와 캡처 카드 사이의 이동을 더욱 직관적으로 개선하고, 검색창 커서가 튀는 현상을 해결했습니다.
- **Svelte 5 안정성 대규모 패치**: 렌더링 도중 상태 오염으로 인한 크래시 현상을 해결하고 백엔드 동기화 로직을 견고하게 다듬었습니다.
- **회귀 방지 테스트 도입**: 핵심 로직에 대한 자동화 테스트를 구축하여 향후 업데이트에서도 동일한 버그가 발생하지 않도록 조치했습니다.

항상 갈피를 응원해 주셔서 감사합니다! 😊

## v1.0.3
### v1.0.3 업데이트 안내 🚀

이번 업데이트에서는 사용자의 편의를 위한 창 드래그 기능을 추가하고 시스템 안정성을 강화했습니다.

- **안정성 개선**: 내부 타입 시스템 오류를 수정하고 전반적인 앱 빌드 안정성을 높였습니다.

항상 갈피를 응원해 주셔서 감사합니다! 😊
