## v1.0.8
### v1.0.8 업데이트 안내 🚀

이번 업데이트에서는 저장된 지식의 연결을 한눈에 파악할 수 있는 **지식 그래프(Graph View)**를 최초로 도입하고, 나만의 앱 환경을 만들 수 있는 **커스텀 배경 시스템**과 더욱 견고해진 **내부 아키텍처**를 통해 갈피(Galpi)의 완성도를 한 단계 더 높였습니다.

#### 🕸️ 지식 그래프(Graph View) 정식 도입
- **연결의 시각화**: 내가 기록한 갈피들이 서로 어떻게 연결되어 있는지 그래프 형태로 한눈에 확인할 수 있습니다.
- **성능 최적화**: 대량의 데이터 세트에서도 부드러운 탐색이 가능하도록 Rust 백엔드 레벨에서 그래프 엔진을 구축했습니다.

#### 🎨 다이내믹 커스텀 배경 시스템
- **개인화된 배경**: 내가 좋아하는 이미지 URL을 활용해 앱의 배경을 나만의 스타일로 바꿀 수 있습니다.
- **살아있는 배경 (Dynamic Sliding)**: 검색어를 입력하면 글자 길이에 따라 이미지가 위아래로 부드럽게 움직이는 동적 효과를 추가하여 '탐색'의 즐거움을 더했습니다.
- **스마트 UI 레이아웃**: 배경 이미지 유무에 따라 상단 바의 투명도가 지능적으로 조절됩니다. 이미지가 없을 때는 시인성이 높은 **진한 불투명(Solid)** 색상으로 자동 전환됩니다.

#### 🏗️ 아키텍처 업그레이드 및 안정화
- **로직 모듈화**: 비대해진 내부 로직을 Action, Handler, Controller 계층으로 완전히 분리하여 앱의 응답 속도와 안정성을 대폭 개선했습니다.
- **워크플로우 엔진 도입**: 복잡한 작업 흐름을 전담 관리하는 시스템을 구축하여 상태 전환 시 발생할 수 있는 오류를 원천 차단했습니다.
- **정교한 사용성 튜닝**: 
  - 앱 전반의 용어를 '갈피'로 통일하여 정체성을 강화했습니다.
  - 가이드 문구에 사용자가 실제 설정한 글로벌 단축키가 실시간으로 반영되도록 개선했습니다.
  - 편집 모드 진입 시 커서 위치를 최적화하는 등 미세한 사용감을 꼼꼼히 다듬었습니다.

## v1.0.7
### v1.0.7 업데이트 안내 🚀

이번 업데이트에서는 앱의 상태 관리 엔진(FSM)을 전면적으로 고도화하여 작업 흐름의 끊김을 없애고, 설정 화면 및 검색 레이아웃을 개편하여 사용자 편의성을 극대화했습니다.

#### 🎨 설정 화면 및 검색 레이아웃 혁신
- **캡처 카드 상단 고정 (Sticky Header)**: 검색 결과를 스크롤 하더라도 캡처 정보가 항상 상단에 고정됩니다. 
  - **신규 캡처**: 새로운 내용을 저장할 때, 캡처된 제목과 이미지를 보면서 어울리는 태그를 검색하거나 참고할 수 있어 맥락이 유지됩니다.
  - **기존 캡처**: 저장된 갈피 조회 시에도 내가 어떤 컨텍스트 안에 있는지 명확히 파악하며 유사 항목을 탐색하는 '앵커' 역할을 합니다.
- **사이드바 설정 내비게이션**: 설정을 일반, 화면, 동작, 태그, 저장소 등으로 명확히 분류하여 필요한 기능을 더 빠르고 직관적으로 찾을 수 있습니다.
- **컨텍스트 타입 드롭다운**: 태그 타입을 직접 타이핑할 필요 없이, DB에서 사용 중인 태그들을 추천받아 클릭 한 번으로 등록할 수 있습니다.

#### 🔄 캡처 흐름 및 상태 관리 엔진(FSM) 고도화
- **똑똑한 재캡처 시스템**: 갈피 삭제 시 해당 정보를 즉시 다시 감지하여 신규 캡처 카드로 자동 전환합니다. 삭제 후 다시 단축키를 누를 필요 없이 바로 이어서 작업이 가능합니다.
- **깜빡임 없는 UI 전환**: 프로세스 ID 정밀 추적 로직을 통해 정보 갱신 중 발생하는 불필요한 창 깜빡임을 제거하고 부드러운 전환을 구현했습니다.
- **정밀한 커서 및 상태 제어**: 
  - 이름 편집(`Cmd + E`) 시 커서가 제목 바로 뒤(태그 시작 전)에 위치하도록 조정하여 수정 편의성을 높였습니다.
  - 편집 중 취소(`Esc`) 또는 저장(`Enter`) 시, 이전 작업 위치(검색 리스트 또는 캡처 뷰)를 정확히 기억하여 복구합니다.
- **스마트 캡처 캔슬**: 캡처 카드 포커스 시 `Esc`를 누르면 창 전체가 닫히는 대신 캡처 카드만 즉시 제거되어 검색 흐름을 방해하지 않습니다.

#### 🛠️ 시스템 안정성 및 디버깅 강화
- **개발 환경 완벽 분리**: 개발용 앱(GalpiDev)과 정식 버전을 완전히 독립시켜, 개발 중에도 정식 앱을 안정적으로 사용할 수 있습니다.
- **상세 에러 리포팅 & 로깅**: 업데이트 실패 시 구체적인 원인을 표시하고, 내부 로그 파일 기록 기능을 강화하여 기술적 이슈 대응력을 높였습니다.
- **ID 타겟팅 버그 해결**: 특정 상황에서 편집 창이 엉뚱한 항목을 타겟팅하거나 실행되지 않던 문제를 해결했습니다.
- **보안 안정화**: 자동 업데이트용 공용 키 설정을 최신화하여 앱의 보안성과 신뢰성을 높였습니다.

### v1.0.6 업데이트 안내 🚀

이번 업데이트에서는 검색 엔진의 정확도를 정밀하게 조정하고, 사용자님의 설정에 맞춰 런처 창의 크기와 위치를 유지하는 기능을 추가했습니다.

#### 📐 창 크기 및 위치 기억 (Dynamic Persisting)
- **개별 맞춤 리스트 설정**: 설정에서 노출될 갈피 개수(6~12개)를 조정하면, 해당 개수에 딱 맞게 런처의 세로 길이가 자동으로 최적화됩니다. 
- **위치 기억 기능**: 드래그 모드를 사용 중일 때, 런처를 이동시킨 위치를 기억합니다. 이제 매번 창을 켤 때마다 중앙으로 복구되지 않고, 최근에 배치한 그 위치에서 바로 나타납니다.
- **메인 메뉴바 상시 구동 최적화**: macOS 하단 Dock에서 아이콘이 노출되지 않도록 개선하여 더욱 깔끔한 상시 구동 환경을 제공합니다.
- **기본 단축키 변경**: 기존 `Option + Space`에서 `Option + Command + G`로 기본 단축키를 변경하여 다른 앱과의 충돌을 방지했습니다.

#### 🔍 검색 및 하이라이트 시스템 고도화
- **검색 정확도 원천 복구**: 가장 안정적이었던 초성 및 영문 검색 로직으로 회귀하여 짧은 키워드도 누란 없이 완벽하게 찾아냅니다.
- **하이브리드 하이라이트**: 백엔드의 정렬 성능과 프론트엔드의 정교한 하이라이트 범위를 결합하여 시각적 명확성을 높였습니다.

#### 🛠️ 시스템 안정성 개선
- **검색 임계값 제거**: 모든 유효한 결과를 노출하여 검색 결과의 신뢰도를 높였습니다.
- **버전 동기화 최적화**: 내부 라이브러리 간의 버전 및 타입 일치를 통해 시스템 안정성을 강화했습니다.

항상 갈피를 더 편리하게 만들어주시는 사용자 여러분의 의견에 감사드립니다! 😊
