# 🔖 갈피(Galpi) - 생각의 연결, 지식의 나침반 (Open Beta)

**갈피**는 흩어진 생각과 지식의 조각들을 연결하여 맥락을 복원하는 스마트 데스크탑 애플리케이션입니다. 현재 **공개 베타 테스트**를 통해 더욱 완벽한 서비스를 준비하고 있습니다.

---

## 📥 설치 및 실행 가이드 (macOS)

### 1단계: 설치 파일 다운로드
[베타 릴리즈 페이지](https://github.com/wisgraph/galpi-release/releases)에서 고객님의 기기 환경에 맞는 파일을 다운로드하세요.
*   **`.dmg` 파일**: 처음 설치하시는 분들을 위한 표준 설치 파일입니다.
*   다운로드 후 실행하여 **galpi** 아이콘을 **Applications(응용 프로그램)** 폴더로 드래그 앤 드롭 하세요.

## 🔗 업데이트 주소
갈피 베타 앱은 아래 경로를 통해 최신 업데이트를 자동 확인합니다.
`https://github.com/wisgraph/galpi-release/releases/latest/download/latest.json`

### 2단계: "확인되지 않은 개발자" 보안 경고 해결법
갈피는 현재 초기 개발 단계로, 실행 시 macOS 보안 정책에 의해 **'확인되지 않은 개발자'** 메시지가 표시될 수 있습니다. 이 경우 아래 방법 중 하나로 실행을 허용해 주세요.

#### 방법 A: 설정에서 직접 허용 (추천)
1.  앱을 실행할 때 경고창이 뜨면 **[취소]**를 누릅니다.
2.  **시스템 설정(System Settings) > 개인정보 보호 및 보안**으로 이동합니다.
3.  하단에 '갈피(galpi)이(가) 차단되었습니다'라는 메시지 옆의 **[확인 없이 열기]** 버튼을 클릭합니다.
4.  지문 또는 암호 인증 후 실행됩니다.

#### 방법 B: 터미널 명령어로 한 번에 해결
설치 후 앱이 절대 열리지 않는 경우, **터미널(Terminal)** 앱을 열고 아래 명령어를 복사하여 붙여넣으세요.
```bash
sudo xattr -rd com.apple.quarantine /Applications/galpi.app
```
*(명령어 입력 후 Mac 로그인 암호를 한 번 입력해 주세요. 입력 시 화면에는 글자가 보이지 않지만 정상적으로 입력되는 중입니다.)*

## 🛠️ 주요 기능 및 지원 환경

### 1. 지능형 문맥 보존 검색
단순히 제목을 찾는 것을 넘어, 내가 작업을 멈췄던 바로 그 순간의 맥락을 함께 검색하고 복원합니다.

### 2. 지원 애플리케이션 (Mac 최적화)
현재 갈피는 업무 효율을 극대화하는 핵심 앱들을 완벽하게 지원합니다:
*   **Web Browsers**: Chrome, Safari, Arc 등 모든 웹페이지의 문맥 저장
*   **Obsididun (옵시디언)**: 제텔카스텐 지식 관리와의 완벽한 연동
*   **Preview (미리보기)**: PDF 및 이미지 문서 작업 흐름 보존
*   **Finder (파인더)**: 로컬 파일 시스템의 탐색 맥락 복원

### 3. 더 똑똑한 사용성
*   **자동 라이선스 활성화**: 이메일 링크 하나로 정식 버전 변환
*   **글로벌 핫키**: `Option + Space`로 언제 어디서나 지식 소환
